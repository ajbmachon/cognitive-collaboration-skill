A Claude Code skill providing twelve cognitive modes for bulletproofing decisions and expanding thinking.

## Installation

```bash
/plugin marketplace add /path/to/cognitive-collaboration-skill
/plugin install cognitive-collaboration@cognitive-collaboration-dev
```

## Cognitive Modes

### CRITIQUE Modes (Stress-test ideas)
- **Red Team** - Attack your own ideas
- **Steelman** - Strengthen opposing arguments
- **Pre-Mortem** - Imagine failure and work backwards
- **Assumptions** - Identify hidden assumptions
- **Biases** - Detect cognitive biases

### EXPANSION Modes (Explore possibilities)
- **First Principles** - Break down to fundamentals
- **Analogist** - Find patterns from other domains
- **Scenario Weaver** - Explore alternative futures
- **Perspective Shifter** - See through other lenses
- **Synthesizer** - Combine disparate ideas
- **Option Generator** - Expand the solution space
- **Values Clarifier** - Align with core values

## Usage

Invoke by asking Claude to use cognitive collaboration modes, or when preparing pitches, making decisions, stress-testing ideas, or exploring alternatives.
